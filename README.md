# Exchange Rates App – Docker Setup

הרצה עם Docker

```bash
docker-compose up --build
```

- אפליקציית ריאקט: http://localhost:3000
- API ב־ASP.NET: http://localhost:5017/api/exchange

הרצה עם vs
beckend -
cd backend
dotnet run

frontend-
npm install
npm run dev
